Generate SQL

Users will soon be able to create or generate SQL queries based on their questions. This feature will utilize SQL models from either Google Gemini or GPT-3 Turbo. Users will have a limited number of requests, but once these are exhausted, they will have the option to provide API keys from OpenAI. These keys will be stored locally, allowing continued use of the models based on their API keys.

NOTE: For security reasons, your API keys will be stored locally. This means you will need to provide them again after logging out, unless you opt-in for cloud storage by agreeing to have us store the keys in a secure database.
